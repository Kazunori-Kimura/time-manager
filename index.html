勤怠管理表 スマートフォンで簡単勤怠管理

毎日の出勤・退勤時間入力を簡易化し、面倒な勤怠管理表の作成をサポートします。

簡単入力

出勤・退勤時にボタンをタップするだけで、時間を記録できます。

月次報告書の作成

PDF出力機能により、毎月の報告書作成の手間を削減します。

アプリについて

開発者の方へ

ソースコードは全て GitHub にて管理しています。

バグ・要望等のご報告はGitHubのプロジェクトページにお願いいたします。
